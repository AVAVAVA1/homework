## 3. 螺旋桨基础理论（Chapter 3）
### 3.1 理想推进器理论（Actuator Disc）
- **轴向诱导速度**：远前方 $0$，盘面 $u_a/2$，远后方 $u_a$。
- **推力**  
  $$T = \rho A_0 \big(V_A + \frac{u_a}{2}\big) u_a$$
- **效率**  
  $$\eta_i = \frac{2}{1+\sqrt{1+\sigma_T}}, \quad \sigma_T = \frac{T}{\frac{1}{2}\rho A_0 V_A^2}$$
  > 记住 $\sigma_T$ 是推力载荷系数；该公式说明载荷越小效率越接近1。

### 3.2 理想螺旋桨理论（考虑旋转）
- 产生周向诱导速度 $u_t$ / 轴向诱导速度 $u_a$。
- 诱导速度正交性（动能定理导出）：
  $$(V_A + \frac{u_a}{2})u_t = u_a(\omega r - \frac{u_t}{2})$$
- 效率表达式：
  $$\eta_i \approx \frac{V_A}{V_A+u_a} \cdot \frac{\omega r - u_t}{\omega r}$$
  实质上是 Rankine-Froude 效率的推广。

### 3.3 叶元体理论
- 二维机翼 **升力** $L = \rho V \Gamma$，**阻力** $D = \frac{1}{2}\rho V^2 S C_D$。
- 对叶元体，将 $L$、$D$ 分解为推力 $dT$ 和旋转力 $dF_\theta$（产生扭矩 $dQ = r dF_\theta$），再沿径向积分并乘以桨叶数。
- 考虑三维效应：自由涡导致下洗速度，从而使攻角减小（需引入诱导速度修正）。
- **重要定性结论**：叶元体理论比动量理论更细致，但要解出实际的推力和扭矩需要知道环量分布（环流理论）。

### 3.4 进程、滑脱、进速系数
- 进速系数 $$J = \frac{V_A}{nD}$$
- 滑脱距离 $$h_P = \frac{V_A}{n}$$
- 螺距 $P$、滑脱 $= P - h_P$、滑脱比 $s = 1 - \dfrac{h_P}{P} = 1 - \dfrac{J}{P/D}$
- 无推力进程 $P_1$（对应 $K_T = 0$）、无转矩进程 $P_2$（对应 $K_Q = 0$）。
- **必考**：给你 $K_T$~$J$、$K_Q$~$J$ 曲线，要能读出 $P_1$、$P_2$ 对应的 $J$，以及滑脱比计算。

## 6. 螺旋桨的空泡现象（Chapter 6）
### 6.1 空泡成因
- 局部压力 $p_b \le p_v$（汽化压力）时水汽化。
- 空泡数： $\sigma = \dfrac{p_0 - p_v}{\frac{1}{2}\rho V_0^2}$  
  减压系数： $\xi = \dfrac{p_0 - p_b}{\frac{1}{2}\rho V_0^2}$
- **空泡发生条件**：$\xi \ge \sigma$ ； $\sigma$ 越小（高速、浅水、高温）越易空泡。

### 6.2 影响 $\xi$ 的因素
- **攻角 $\alpha$ ↑** → 升力↑ → 压力峰变尖 → $\xi_{\max}$ ↑ → 更易空泡。
- **切面形状**：弓型切面压力分布平坦 → $\xi_{\max}$ 较小 → 抗空泡性能优于机翼型。
- **沉深 ↑ / 转速 ↓** → 静压↑ → $\sigma$ ↑ → 抗空泡。

### 6.3 空泡的两个阶段
| 阶段 | 特征 | 对性能影响 |
|------|------|------------|
| 第一阶段 | 局部空泡 | $K_T, K_Q, \eta_0$ 基本不变；可能剥蚀 |
| 第二阶段 | 空泡延伸至随边外，桨叶表面全泡 | 性能显著下降；振动、噪声加剧 |

### 6.4 空泡斗（Cavitation Bucket）
- 以攻角 $\alpha$ 为横轴，空泡数 $\sigma$ 为纵轴，形成“无空泡区”的斗形边界。
- 桨叶设计时要使各叶切面的工作点落在无空泡区内。

### 6.5 延缓空泡的措施
1. 增加盘面比 $A_E/A_0$（降低单位面积载荷）
2. 采用弓型或均匀压力分布的切面
3. 减小叶根附近的螺距
4. 增加螺旋桨沉没深度
5. 降低转速

### 6.6 空泡校核（Burrill 限界线）
计算 **0.7R 处的空泡数**和**单位投射面积平均推力系数**，查 Burrill 图谱判断是否在限界线之下。

$$\sigma_{0.7R} = \frac{p_0 - p_v}{\frac{1}{2}\rho V_{0.7R}^2}, \quad V_{0.7R} = \sqrt{V_A^2 + (0.7\pi n D)^2}$$
$$\tau_c = \frac{T/A_P}{\frac{1}{2}\rho V_{0.7R}^2}$$

## 7. 螺旋桨的强度校核（Chapter 7）
- **任务**：确保桨叶能承受最大载荷（离心力+推力弯矩），确定合理厚度分布。
- **两种方法**：
  - **规范校核**（依据船级社规范，给定最小要求厚度）
  - **分析计算**（悬臂梁或有限元，考虑离心力和流体力）
- **关键控制截面**：0.25R（叶根）和 0.6R 处的厚度必须大于规范要求。
- **厚度分布**：可线性分布、非线性分布，或按荷兰船池推荐的分布。
- **螺距修正**：为满足强度增厚时，叶切面形状改变，水动力性能会偏移，因此需要小幅调整螺距，使原设计的推力和扭矩特性基本不变。
- **重量与惯性矩计算**：用于轴系设计、振动分析和加工评估。

> 本章通常考**简答**：为什么要进行螺距修正？或解释强度校核的基本思路。

## 8. 螺旋桨图谱设计（Chapter 8）
### 8.1 设计问题分类
- **初步设计**：给定 $V$, $P_E$，选定 $D$（或 $n$），确定最佳 $n$（或 $D$）、$P/D$、$\eta_0$、$P_S$。
- **终结设计**：给定 $P_S$, $n$, $P_E$ 曲线，求最大航速 $V$，以及对应的 $D$, $P/D$, $\eta_0$。

### 8.2 B‑δ 设计图谱
- 两个基本系数（**必背**）：
  $$B_P = \frac{N \sqrt{P_D}}{V_A^{2.5}} \quad (\text{注意 }N\text{ rpm}, P_D\text{ hp}, V_A\text{ kn})$$
  $$\delta = \frac{30.86 N D}{V_A}$$
- **图谱用法**：由 $B_P$ 沿等效率曲线查到 $\eta_0$ 最大点 → 读取 $\delta$, $P/D$, $\eta_0$ → 反算 $D$ 和所需主机功率。

### 8.3 船‑桨‑机匹配
- 阻力发生变化时，螺旋桨的工作点沿特性曲线移动；校核曲线（航行特性）用于检验主机是否过载。
- 系泊状态（$V_A=0$, $J=0$）推力最大，转矩也最大，需校核主机能否承受。

## 计算题专项突破（Exam‑type Problems）
根据作业题和课件，常考以下题型：

### 题型 A：敞水性征曲线计算与读图
**给定**：$D$, $n$, $\rho$, 及 $V_A$, $T$, $Q$ 数据  
**求解**：
1. $J = \dfrac{V_A}{nD}$，注意 $n$ 用 rps。
2. $K_T = \dfrac{T}{\rho n^2 D^4}$, $K_Q = \dfrac{Q}{\rho n^2 D^5}$
3. $\eta_0 = \dfrac{J}{2\pi}\dfrac{K_T}{K_Q}$
4. 绘制 $K_T$-$J$, $K_Q$-$J$, $\eta_0$-$J$ 曲线。

**读图**（给曲线）：
- $J$ 已知 → 查 $K_T, K_Q$ → 得 $T, Q, \eta_0$。
- 滑脱比 $s = 1 - \dfrac{J}{P/D}$。
- 零推力进程 $P_1 = J_1 \cdot D$（查 $K_T=0$ 的 $J$）。
- 零转矩进程 $P_2 = J_2 \cdot D$（查 $K_Q=0$ 的 $J$）。
- 最大效率进速：对应 $\eta_0$ 最高点的 $J$。

### 题型 B：功率传递链计算（常考）
**步骤**：  
1. 由 $R$、$V$ 计算 $P_E = R \cdot V$（注意单位：马力 $= \text{kgf} \cdot \text{m/s} / 75$，1 kn = 0.5144 m/s）。
2. 由伴流 $w$ 得 $V_A = V(1-w)$，推马力 $P_T = T V_A$。
3. 转矩 $Q$ 已知（或由 $P_D$ 反求），计算 $P_D = 2\pi n Q$。
4. 依次求：
   - $\eta_H = \frac{1-t}{1-w}$  （或直接 $t = (T-R)/T$）
   - $\eta_R = 1/i_2$（或按定义 $Q_0/Q_B$）
   - $\eta_B = P_T/P_D$
   - $\eta_0 = \eta_B / \eta_R$
   - $\eta_D = \eta_H \eta_0 \eta_R = P_E/P_D$
5. 验证 $P_E$ 与 $P_D$ 的关系。

### 题型 C：系柱推力（Bollard Pull）
**条件**：$V_A=0$ ($J=0$)，保持原转矩 $Q$ 不变。  
1. 查敞水曲线 $J=0$ → 得 $K_{T0}$、$K_{Q0}$。
2. 由原工况求得转矩 $Q$（如 $Q = \frac{75 \cdot \text{DHP}}{2\pi n}$）。
3. 系柱时，由 $Q = K_{Q0} \rho n'^2 D^5$ 解出新转速 $n' = \sqrt{\dfrac{Q}{K_{Q0}\rho D^5}}$。
4. 新推力 $T_0 = K_{T0} \rho n'^2 D^4$。
5. 有效推力 $T_e = T_0(1-t)$（系柱时推力减额通常不同于直航，但题目多直接给定或假定不变）。

### 题型 D：临界雷诺数判断
计算 $Re_{0.75R}$，并与 $3\times10^5$ 比较。  
$$Re = \frac{b_{0.75R} \sqrt{V_A^2 + (0.75\pi n D)^2}}{\nu}$$
- 注意 $V_A = J n D$，$n = N/60$ rps。

### 题型 E：空泡校核（Burrill 限界）
计算 $\sigma_{0.7R}$ 和 $\tau_c$，与图谱比较。若点落在限界线下方则安全。

## 最后叮嘱
- **公式必须能默写**：$K_T$, $K_Q$, $J$, $\eta_0$, $\eta_H$, $\eta_D$, $\sigma$, $\xi$, $B_P$, $\delta$, $Re$。
- **单位要警惕**：$n$ 用 rps，$V$ 用 m/s 或 kn，$P$ 用 hp 或 W。
- **概念题**：推进器种类、叶切面名称、空泡斗、尺度效应、伴流与推力减额的物理含义、螺旋桨设计两类问题的区别等。
- **临场技巧**：计算题先列清楚已知量，再写出需要哪些中间变量，按定义一步步算，检查量纲。
