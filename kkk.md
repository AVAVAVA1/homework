船舶推进考前速通复习提纲（按章节 + 计算专项）

时间紧任务重——这份提纲帮你把8章核心内容压缩成“必考概念 + 必背公式 + 典型解法”，并搭配高频计算题型拆解。请按顺序快速过一遍，确保公式能默写、定义能复述。闭卷考试，沉着作答！

1. 概述

· 推进器种类：螺旋桨、风帆、明轮、直叶推进器、喷水推进器等；螺旋桨因结构简单、效率较高而应用最广。
· 功率定义
    $P_E$ （有效马力，克服船体阻力）
    $P_D$ （螺旋桨收到马力）
    $P_S$ （主机发出马力）
· 效率公式
    轴系传送效率 $\eta_s = P_D/P_S$
    推进效率（初定义） $\eta_D = P_E/P_D$
    推进系数 $PC = \eta_s \cdot \eta_D = P_E/P_S$

重点：能根据题目直接计算 $P_E = R \cdot V$，再配合效率求 $P_D$、$P_S$。

2. 螺旋桨几何特征

· 结构：桨叶、桨毂、毂帽；常见三叶、四叶、五叶。
· 桨叶术语：叶面、叶背、导边、随边、叶梢、叶根。
· 叶切面参数：拱度、厚度、弦长、螺距 $P$ / 螺距角、纵斜、侧斜。
· 面积比（题目中常给出）：
  $A_P/A_0$（投射面比）、$A_E/A_0$（伸张面比）、$A_D/A_0$（展开面比）、盘面比（常指 $A_E/A_0$）

考法：通常作为概念题出现，让你识别叶切面或解释面积比。

3. 螺旋桨基础理论

3.1 理想推进器理论（Actuator Disc）

· 轴向诱导速度：远前方 $0$，盘面 $u_a/2$，远后方 $u_a$
· 推力：$T = \rho A_0 (V_A + u_a/2) u_a$
· 效率：$\eta_i = \frac{2}{1+\sqrt{1+\sigma_T}}, \quad \sigma_T = \frac{T}{\frac{1}{2}\rho A_0 V_A^2}$

记住 $\sigma_T$ 是推力载荷系数；该公式说明载荷越小效率越接近1。

3.2 理想螺旋桨理论（考虑旋转）

· 产生周向诱导速度 $u_t$ / 轴向诱导速度 $u_a$
· 诱导速度正交性：$(V_A + u_a/2)u_t = u_a(\omega r - u_t/2)$
· 效率表达式：$\eta_i \approx \frac{V_A}{V_A+u_a} \cdot \frac{\omega r - u_t}{\omega r}$

3.3 叶元体理论

· 升力 $L = \rho V \Gamma$，阻力 $D = \frac{1}{2}\rho V^2 S C_D$
· 将 $L$、$D$ 分解为推力 $dT$ 和旋转力，再沿径向积分并乘以桨叶数
· 需考虑自由涡导致的下洗速度

3.4 进程、滑脱、进速系数

· 进速系数：$J = \frac{V_A}{nD}$
· 滑脱比：$s = 1 - \frac{J}{P/D}$
· 无推力进程 $P_1$（对应 $K_T = 0$）、无转矩进程 $P_2$（对应 $K_Q = 0$）

必考：给你 $K_T$-$J$、$K_Q$-$J$ 曲线，要能读出 $P_1$、$P_2$ 对应的 $J$，以及滑脱比计算。

4. 螺旋桨模型的敞水试验

4.1 相似条件

相似类型 条件 实际处理
几何相似 缩尺比 $\lambda$ 严格相似
运动相似 $J_m = J_s$ 保持相等
动力相似（Fr） Fr相等 桨模浸深 $h_s > 0.625D$
动力相似（Re） $Re > Re_{\text{临界}}$ $Re_{\text{临界}} = 3\times 10^5$

· 雷诺数定义（0.75R处）：$Re_{0.75R} = \frac{b_{0.75R} \sqrt{V_A^2 + (0.75\pi n D)^2}}{\nu}$
· 尺度效应：同 $J$ 下，$K_{Tm} < K_{Ts}$，$K_{Qm} > K_{Qs}$，$\eta_{0m} < \eta_{0s}$

4.2 敞水性征曲线

· 无因次系数：
  $K_T = \frac{T}{\rho n^2 D^4}$
    $K_Q = \frac{Q}{\rho n^2 D^5}$
    $\eta_0 = \frac{J}{2\pi} \cdot \frac{K_T}{K_Q}$

考法：给定试验数据 $V_A$、$T$、$Q$，计算 $K_T$、$K_Q$、$\eta_0$ 并绘制曲线。

5. 螺旋桨与船体相互作用

5.1 伴流

· 成因：势伴流、摩擦伴流、波浪伴流
· 伴流分数：$w = \frac{V - V_A}{V}$，$V_A = V(1-w)$
· 标称伴流 vs 实效伴流（等推力法确定）

5.2 推力减额

· 推力减额分数：$t = \frac{T - R}{T}$
· 有效推力：$T_e = T(1-t) = R$（直航状态）

5.3 功率传递与效率链 ⭐考试核心

功率流：
$P_S$ →($\eta_s$)→ $P_D$ →($\eta_R$)→ $P_{D0}$ →($\eta_0$)→ $P_T$ →($\eta_H$)→ $P_E$

效率 表达式 说明
轴系传送效率 $\eta_s = P_D/P_S$ 主机到桨轴
相对旋转效率 $\eta_R = Q_0/Q_B$ 伴流不均匀导致扭矩变化
敞水效率 $\eta_0 = \frac{T V_A}{2\pi n Q_0} = \frac{J K_T}{2\pi K_Q}$ 桨模敞水试验所得
船身效率 $\eta_H = \frac{1-t}{1-w}$ 船体与桨相互增益
推进效率 $\eta_D = P_E/P_D = \eta_H \eta_0 \eta_R$ 也称为 QPC
推进系数 $PC = \eta_s \eta_D = P_E/P_S$ 总效率

💡 一句话记忆：整个链就是 $P_E = P_S \cdot \eta_s \cdot \eta_R \cdot \eta_0 \cdot \eta_H$

6. 螺旋桨的空泡现象

6.1 空泡成因

· 局部压力 $p_b \le p_v$（汽化压力）时水汽化
· 空泡数：$\sigma = \frac{p_0 - p_v}{\frac{1}{2}\rho V_0^2}$
· 减压系数：$\xi = \frac{p_0 - p_b}{\frac{1}{2}\rho V_0^2}$
· 空泡发生条件：$\xi \ge \sigma$；$\sigma$ 越小越易空泡

6.2 空泡两个阶段

阶段 特征 对性能影响
第一阶段 局部空泡 $K_T, K_Q, \eta_0$ 基本不变；可能剥蚀
第二阶段 空泡延伸至随边外 性能显著下降；振动、噪声加剧

6.3 延缓空泡的措施

1. 增加盘面比 $A_E/A_0$
2. 采用弓型切面
3. 减小叶根附近的螺距
4. 增加螺旋桨沉没深度
5. 降低转速

6.4 空泡校核（Burrill 限界线）

计算0.7R处的空泡数和单位投射面积平均推力系数：
\sigma_{0.7R} = \frac{p_0 - p_v}{\frac{1}{2}\rho V_{0.7R}^2}, \quad V_{0.7R} = \sqrt{V_A^2 + (0.7\pi n D)^2}


\tau_c = \frac{T/A_P}{\frac{1}{2}\rho V_{0.7R}^2}

7. 螺旋桨强度校核

· 关键控制截面：0.25R（叶根）和 0.6R 处厚度必须大于规范要求
· 两种方法：规范校核（船级社规范）和分析计算（悬臂梁或有限元）
· 螺距修正：为满足强度增厚时，需小幅调整螺距保持水动力性能基本不变
· 本章通常考简答：为什么要进行螺距修正？或解释强度校核的基本思路

8. 螺旋桨图谱设计

8.1 设计问题分类

· 初步设计：给定 $V$、$P_E$，选定 $D$（或 $n$），确定最佳参数
· 终结设计：给定 $P_S$、$n$、$P_E$ 曲线，求最大航速 $V_{max}$

8.2 B-δ 设计图谱

两个基本系数（必背）：
B_P = \frac{N \sqrt{P_D}}{V_A^{2.5}} \quad (N\text{ rpm}, P_D\text{ hp}, V_A\text{ kn})


\delta = \frac{30.86 N D}{V_A}

由 $B_P$ 沿等效率曲线查到 $\eta_0$ 最大点 → 读取 $\delta$、$P/D$、$\eta_0$ → 反算 $D$ 和所需主机功率

计算题专项突破

题型A：敞水性征曲线计算与读图

1. $J = \frac{V_A}{nD}$（注意 $n$ 用 rps）
2. $K_T = \frac{T}{\rho n^2 D^4}$，$K_Q = \frac{Q}{\rho n^2 D^5}$
3. $\eta_0 = \frac{J}{2\pi} \cdot \frac{K_T}{K_Q}$
4. 读图：查 $P_1$（$K_T=0$ 的 $J$）、$P_2$（$K_Q=0$ 的 $J$）、滑脱比、最大效率点

题型B：功率传递链计算 ⭐常考

1. $P_E = R \cdot V$（注意单位换算：1 kn = 0.5144 m/s）
2. $V_A = V(1-w)$
3. $P_D = 2\pi n Q$
4. 依次求 $\eta_H = \frac{1-t}{1-w}$，$\eta_R$，$\eta_0$，$\eta_D = P_E/P_D$
5. 验证 $P_E$ 与 $P_D$ 的关系

题型C：系柱推力

1. 查敞水曲线 $J=0$ → 得 $K_{T0}$、$K_{Q0}$
2. 由 $Q = K_{Q0} \rho n'^2 D^5$ 解出新转速 $n'$
3. 新推力 $T_0 = K_{T0} \rho n'^2 D^4$
4. 有效推力 $T_e = T_0(1-t)$

题型D：临界雷诺数判断

计算 $Re_{0.75R} = \frac{b_{0.75R} \sqrt{V_A^2 + (0.75\pi n D)^2}}{\nu}$，与 $3 \times 10^5$ 比较

题型E：空泡校核

计算 $\sigma_{0.7R}$ 和 $\tau_c$，与 Burrill 图谱比较，点落在限界线下方则安全
